# Daniel J Profile Card

A responsive and accessible Profile Card built with HTML, CSS, and JavaScript.

## Features

- Semantic HTML structure
- Fully accessible (ARIA + keyboard navigation)
- Responsive design (mobile, tablet, desktop)
- Dynamic time display (Date.now in milliseconds)
- Data-testid attributes for automated testing
